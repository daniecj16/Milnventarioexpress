# Milnventarioexpress
 **Desarrollado por:** Daniel Cruz ** Actividad Unidad 1Descripción **MiInventarioExpress** es una aplicación web desarrollada con Node.js, Express y MongoDB que permite gestionar productos de forma eficiente. Incluye autenticación de usuarios, carga de imágenes, validación de formularios y un módulo de chat en tiempo real para administradores.
